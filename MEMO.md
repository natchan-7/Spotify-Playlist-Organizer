# MEMO

## 今後追加したい機能

- 複数タグを組み合わせてプレイリストを作れるようにする
- タグごとの曲数を一覧で見やすく表示する
- 作成済みプレイリストの履歴をアプリ内で見られるようにする
- よく使うタグを上に出すなど、タグ検索の並び順を改善する

## 改善していきたい文言

- 自動タグが 0 件だったときの説明を、もう少しやさしく分かりやすくする
- Spotify 側の制限で失敗したときの文言を、利用者向けに簡潔にする
- 保存データまわりの説明を、削除してよいものだと直感的に分かる表現にする
- プレイリスト作成成功時の案内を、もう少し達成感のある言い回しにする

## やってみたいこと

- auto tag の精度をさらに上げる
- Spotify のジャンル情報だけに頼らない補助ルールを増やす
- UI 全体の細かいアニメーションや反応を整える
- 共有公開しやすい形にできるかをあらためて検証する
